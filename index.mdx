MockK is the go-to mocking library for Kotlin. It provides a clean, expressive DSL that feels natural to Kotlin developers — with first-class support for coroutines, final classes, Android, and more.

```kotlin
val car = mockk<Car>()

every { car.drive(Direction.NORTH) } returns Outcome.OK

car.drive(Direction.NORTH)

verify { car.drive(Direction.NORTH) }
confirmVerified(car)
```

## Why MockK?

MockK was built from the ground up for Kotlin. Unlike Java-first mocking libraries, it understands Kotlin's type system, coroutines, and language features.
